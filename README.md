# GenerativeAI

My GenAI journey — a living repository of experiments where I explore different types of LLMs and the practical ways to integrate them into real applications.

This repo captures hands-on learning across:
- **Prompting patterns** (structured prompts, prompting strategies, reliability)
- **Retrieval-Augmented Generation (RAG)** (chunking, embeddings, vector search, grounding)
- **Tool / function calling** (connecting LLMs to APIs, tools, and workflows)
- **Agents & orchestration** (multi-step reasoning, planning, tool-use loops)
- **Evaluation** (quality checks, regression tests, hallucination/grounding checks)
- **Framework nuances** (what changes between frameworks/providers, trade-offs, and common pitfalls)

The goal is to document what I build and what I learn—especially the *nuances* that matter in production: context limits, latency, cost, prompt/model sensitivity, and failure modes.
